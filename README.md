# Bus Guidance System

A Java application that models a bus route management and scheduling system, built with Maven and validated through a comprehensive JUnit test suite.

## What it does

- Manages bus routes, schedules, and driver assignments using clean object-oriented design.
- Models real-world transport logic — routes, stops, timing, and driver-bus relationships — as a set of cooperating Java classes.
- Includes integration tests covering core scheduling and validation logic to catch edge cases before they reach production.

## Skills demonstrated

- Object-oriented design and architecture in Java
- Unit and integration testing with JUnit
- Build management and dependency handling with Maven
- Collaborative development — built and tested across multiple contributors

## Tech stack

Java, Maven, JUnit

## Running it

```bash
mvn clean install
mvn test
```
